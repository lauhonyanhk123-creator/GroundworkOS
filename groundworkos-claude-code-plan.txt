   A server client using createServerClient from @supabase/ssr

   Supabase session refresh logic for the proxy layer
   Contains an updateSession() function that:
   - Creates a Supabase client from request cookies
   - Calls auth.getUser() to refresh the session
   - Returns a mutable response object with updated cookies

   were applied to — never return a new NextResponse after
   setting cookies or the session will not persist

4. proxy.ts in the project root (NOT middleware.ts)

   Example structure:
   export async function proxy(request: NextRequest) { ... }

   Redirects unauthenticated users to /login
